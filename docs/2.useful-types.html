Some things you can do with lists: ```python # list of ingredients salsa_verde = ['tomatillos', 'serrano chile', 'garlic', 'onion', 'cilantro', 'kosher salt'] # how many ingredients? len(salsa_verde) # grab the salt salsa_verde[-1] # pass the pepper mild_salsa = salsa_verde[0 : salsa_verde.index('serrano chile')] + salsa_verde[salsa_verde.index('serrano chile') + 1 : ] # mix it all up import random random.shuffle(mild_salsa) mild_salsa # straighten it up and flip it in reverse mild_salsa.sort() mild_salsa.reverse() # treble the recipe mild_salsa = mild_salsa * 3 ``` --- class: center, middle # Tuples --- # Tuples Good for grouping of dissimilar type values together into one immutable bundle. --- class: center, middle # Other Sequence types --- # Other Sequence types - ranges - dict_keys - dict_values - strings --- class: center, middle # Dictionaries --- # Dictionaries Key->value pairs. --- class: center, middle # Sets --- # Sets Nice to have. --- class: center, middle # Neverending arguments --- # Neverending arguments The * is bright. --- class: center, middle # Functions of the finest order --- # Functions of the finest order Functions can be passed around the table, like any other data. --- class: center, middle # Lambdas --- # Lambdas Sounds like Greek. --- class: center, middle # Decorators --- # Decorators To make your windows pretty. --- class: center, middle # Conclusions --- # Conclusions Thank you.
